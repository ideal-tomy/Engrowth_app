# 1PR粒度の実装バックログ

## Goal

- 実装を1PR=1目的で進められるバックログへ分解する。
- DB変更が必要な項目を抽出し、実行順・影響を記述する。

## Non-Goal

- 本バックログの「実行」は別タスク。ここでは計画・分解のみ。

## Exit Criteria

- 各項目が1PRで着手可能な粒度になっている。
- 依存関係・DB変更順が明記されている。
- 高優先機能（チュートリアル離脱改善、ロール導線）の仕様が確定している。
