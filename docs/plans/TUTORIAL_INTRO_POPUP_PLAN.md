# 学習説明ポップアップ実装PLAN

## Goal
指オーバーレイ完了後、学習画面へ遷移する前に「どんな学習か」「何をすればいいか」を説明するポップアップを挿入し、初見ユーザーの不安を減らす。

## Non-goal
- 既に学習画面にいる通常導線での説明表示（今回はチュートリアル導線のみ）
- 動画や複数ステップのチュートリアル（シンプルな1枚ポップアップ）

## Exit Criteria
- 30秒会話・パターンスプリント・3分会話の3つすべてで、指オーバーレイ完了後に説明ポップアップが表示される
- ユーザーがCTAをタップしたらポップアップが閉じ、該当学習画面へ遷移する
- engrowth_theme 準拠の見た目

## 影響ファイル
- `lib/widgets/tutorial/learning_intro_dialog.dart`（新規）
- `lib/screens/scenario_learning_screen.dart`
- `lib/screens/pattern_sprint_list_screen.dart`
- `lib/screens/story_training_screen.dart`

## 説明文言（案）

| 学習 | 説明文 | CTA |
|------|--------|-----|
| 30秒会話 | 30秒前後の会話のやりとりを体験します。全体の会話を聴いた後、A役（B役）を選択して会話の練習を行います。 | 会話全体を聴く |
| パターンスプリント | 聞く→まねして言うを短時間で繰り返します。音声を聴いたら即座にリピートしましょう。 | スタート |
| 3分会話 | 約3分の英会話ストーリーを体験します。会話の流れに沿って、聞いたり話したりを繰り返していきます。 | はじめる |

## フロー
1. 指オーバーレイがタップ演出を完了
2. 説明ポップアップをフェードイン表示（400ms easeOut）
3. 約5秒後に自動で閉じ、該当学習画面へ push 遷移
4. （任意）タップで早期閉じ・遷移も可能

※ ユーザーアクションを増やさない設計。ボタンは廃止し、読む時間を考慮した自動閉じに統一。
