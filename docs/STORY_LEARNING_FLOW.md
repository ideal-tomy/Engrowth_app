# 3分会話 学習フロー（ポップアップカルーセル）

## 概要

一覧でカードをタップ → ポップアップ内で学習 → 「次の学習へ」で次ストーリーへ。ページ遷移せずカルーセル内で完結する。

## カルーセルのページ構成（次ストーリーありの場合）

| Index | 内容 |
|-------|------|
| 0 | 現在ストーリーの学習（StoryStudyScreen） |
| 1 | 「次の学習へ」メッセージ（3秒で自動送り or 「次へ」タップ） |
| 2 | 次ストーリーのカード＋「学習する」ボタン |
| 3 | 次ストーリーの学習（StoryStudyScreen） |

## 「次の学習へ」を押したあとの流れ（簡略化）

1. **6択ポップアップで「次の学習へ」をタップ**
   - 6択が閉じる → **「[次ストーリーのタイトル] へ進みます」** のポップアップが1つだけ表示される。

2. **3秒後 に自動で閉じる**
   - カルーセルが **次ストーリーの学習画面まで一気に進む**（中間の「次へ」「学習する」のタップは不要）。

3. **次ストーリーの学習画面**
   - 「まずは音声を…」ポップアップは出さず、**再生ボタンが最初から表示**される。
   - **再生ボタンをタップ** → 音声が流れる（iPhone の自動再生制限を避けるため、ユーザーのタップで開始）。

## デバッグ用ショートカット（DEBUG ビルドのみ）

- **「会話を完了状態にする」ボタン**（カルーセル内の学習画面の左上）
- **「完了にする」**（通常のストーリー学習画面の AppBar）

タップすると「聴き終わり」状態になり、6択ポップアップ（A役／B役／もう一度聴く／次の学習へ／学習進捗ページへ）が表示される。音声を最後まで聴かずに「次の学習へ」の流れを確認できる。
