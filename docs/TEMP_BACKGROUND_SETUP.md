# 仮背景画像の設定

## 概要

全シチュエーション画面で使用する仮設定の背景画像です。
アプリの使用イメージを確認するために使用します。

## ファイル配置

以下のパスに画像を配置してください：

```
assets/images/temp_bg.png
```

- **会話学習画面** (`ConversationStudyScreen`): カフェ・レストラン等のシチュエーション背景
- **シナリオ学習画面** (`ScenarioStudyScreen`): 同上

## 画像がない場合

画像が存在しない場合、グレーの背景が表示されます（`errorBuilder` によるフォールバック）。

## 本番用への切り替え

各会話・シナリオに専用画像を設定する場合は、`conversation.thumbnailUrl` やシナリオの画像URLを使用するよう実装を変更してください。
