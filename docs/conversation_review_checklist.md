# 英会話レビューチェックリスト

既存発話を OK / 要修正 / 全面改稿 に仕分ける際の実用チェックリストです。

---

## 1. 自動検出パターン（スクリプトでフラグ可能）

以下の文字列・パターンが含まれる発話は **要修正** または **全面改稿** の候補とする。

| パターン | 説明 | 想定修正 |
|----------|------|----------|
| `establish` + (`user`\|`account`\|`access`) | 「設定」の誤用 | set up, create |
| `direction` + (`is`\|`to`\|`for`) | 「手順・指示」の誤用 | instructions, steps |
| `reveal` + (`onto`\|`on`\|`the`) | 「表示する」の誤用 | show, display |
| `Production\.` \| `production`（文頭・単独） | 文脈外の「生産」 | 削除または適切な語へ |
| `perform rest` \| `I perform rest` | 「休息する」の不自然な直訳 | get some rest, rest up |
| `weight of symptoms` | 「症状の重さ」の直訳 | severity of symptoms |
| `Treat\.` \| `Discover\.` \| `Trip\.` \| `Evening\.`（単独・羅列） | 単語羅列 | 完整な文へ再構成 |
| 連続する短い句のみ（5語以内が3つ以上、接続詞なし） | 文として未成立 | 再構成 |

---

## 2. 目視チェック項目

各発話について、以下を確認する。

### 文法・構成
- [ ] 主語・述語が明確か
- [ ] 時制が場面に合っているか
- [ ] 代名詞の参照が曖昧でないか

### 自然さ
- [ ] ネイティブが日常で言い得るか
- [ ] 堅すぎ・砕けすぎていないか（シーンに応じて）
- [ ] 冗長な重複表現がないか

### 学習効果
- [ ] 初級（1000語）または中級（3000語）で理解可能か
- [ ] 音読しやすい長さか（20語以内推奨）
- [ ] 他シーンへ転用可能なチャンクが含まれているか

---

## 3. シーン別の追加チェック

| シーン | 追加チェック |
|--------|---------------|
| 病院 | 医学用語の過度な使用、症状説明の具体性 |
| 銀行 | 専門用語（口座種類、手数料）の説明が初級者に分かるか |
| ビジネスメール・プレゼン | 文語と口語の混在がないか、フォーマルさの一貫性 |
| カフェ・レストラン | 注文の流れが現実的か、アレルギー説明が自然か |

---

## 4. 分類時のメモ例

仕分け時に `notes` に記入する例。

- `REVISE: establish→set up, direction→instructions`
- `REWRITE: word salad, no complete sentence`
- `OK: natural, matches rubric`

---

## 5. 検証スクリプトの利用

`scripts/validate_conversation_utterances.dart` を実行すると、上記の自動検出パターンに該当する発話をレポートする。出力を `audit_result.csv` の一次入力として利用可能。
