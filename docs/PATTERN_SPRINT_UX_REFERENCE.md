# パターンスプリント UX リファレンス — 他ページへの適用ガイド

**Cursor / AI 向け**: パターンスプリントで実装した「ユーザーアクション最小化・流れるような学習体験」のコンセプトと実装パターンをまとめたリファレンス。3分会話・シナリオ会話など他ページへ同様の体験を反映する際の一次資料。

---

## 1. 依頼時のテンプレート（コピペ用）

```
パターンスプリントの実装内容とコンセプトを基に、3分会話（またはシナリオ会話）にも同様の学習体験を提供できるUI/UXを実現したい。

参照ドキュメント: docs/PATTERN_SPRINT_UX_REFERENCE.md
```

---

## 2. コアコンセプト

| コンセプト | 内容 |
|-----------|------|
| **ユーザーアクション最小化** | 音声再生以外の操作を原則不要に。テキストで「〇〇してください」と促さず、自然の流れで把握できるようにする。 |
| **流れるような展開** | ページ切り替えより、テキストが消えていく・ポップアップが自動で切り替わる演出。目線の移動を最小化。 |
| **行動要請の排除** | 「選択する」「次へ」などのボタンを挟まず、必要な案内は自動表示・自動切り替えで済ませる。 |
| **固定レイアウト** | コンテンツの増減で要素がずれない。エリアを固定し、テキストはフェードで消える演出。 |

---

## 3. 実装パターン一覧

### 3.1 エントリーフロー（選択ポップアップ）

**現状**: 一覧ページを表示 → ユーザーが選択肢を探す

**パターンスプリントの実装**:
- ページ訪問時、最初に**ラージサイズのポップアップ**を表示
- 「どの〇〇でトレーニングしますか？」＋**一覧を1画面で同時表示**（「選択する」ボタンなし）
- タップで即セッション開始

**参照ファイル**: `lib/widgets/pattern_sprint_selection_dialog.dart`

**ポイント**:
- `EngrowthPopupTokens.largeHeightFraction` でサイズ固定
- `AnimatedBackdrop` で背景ぼかし
- 2ステップ（タイトル→一覧）にしない。1画面で完結。

---

### 3.2 セッションダイアログ（学習中の表示）

**レイアウト**:
- **固定エリア**: 英語エリア / 日本語エリア / ボタンエリアを固定高さで配置
- **テキストが消えていく演出**: `AnimatedOpacity` でフェード。ページ切り替えではない。
- **縦方向のバランス**: 日本語を縦中央、ボタンを下30％付近に。`Spacer` で余白調整。
- **ポップアップサイズ**: `EngrowthPopupTokens.largeHeightFraction` で固定（テキスト量に依存しない）

**フェーズ表示**:
- Phase 1: 英日表示
- Phase 2: 日本語のみ表示
- Phase 3: 音声再生アイコン（`_PulsingAudioIcon`）＋ボタン

**排除した要素**:
- 「音声をよく聴いてください」などの常時表示テキスト
- 「音を真似しながら…」などの説明文

**参照ファイル**: `lib/screens/pattern_sprint_session_dialog.dart`

---

### 3.3 ListenFirstPopup の表示制御

**表示条件**:
- パターンスプリントを**初めて開いたときのみ**表示
- 「もう1セット」で始まる新セッションでは**表示しない**
- アカウント作成済み: 「今後このポップアップは表示しない」チェックで永続非表示
- アカウント未作成: 1日1回、初回訪問時のみ表示

**参照ファイル**:
- `lib/widgets/guided_flow/listen_first_popup.dart`
- `lib/services/listen_first_popup_prefs_service.dart`
- `lib/screens/pattern_sprint_list_screen.dart`（`_skipListenFirstForRestart`, `_skipListenFirstForSelectionPopup`）

---

### 3.4 一覧表示のタップで再生

**現状**: 右の再生ボタンのみで再生開始

**パターンスプリントの実装**:
- **テキスト・左の再生アイコン・右の再生ボタン**をタップ → 再生開始
- **ハート（お気に入り）のみ**タップで再生しない

**参照ファイル**: `lib/screens/pattern_sprint_list_screen.dart`（ListTile の `onTap`）

---

### 3.5 カテゴリヘッダーのレイアウト

**現状**: 2カラム（タイトル左・スタートボタン右）

**パターンスプリントの実装**:
- 1行目: アイコン | タイトル | サブテキスト（タイトルの右側）
- 2行目: スタートボタン（1カラム・全幅）
- 一覧のパターン名は**太字**で目立たせる

---

## 4. 技術スタック・共通資産

| 資産 | 用途 |
|------|------|
| `EngrowthPopupTokens.largeHeightFraction` | ポップアップ高さ固定（0.85） |
| `AnimatedBackdrop` | 背景ぼかし |
| `EngrowthPopup` / `EngrowthPopupVariant.missionClear` | ミッション達成ポップアップ |
| `AnimatedOpacity` | テキストのフェード演出 |
| `AnimatedSwitcher` | 日本語↔アイコンの切り替え |
| `lib/theme/engrowth_theme.dart` | トークン・イージング |

## 5. 3分会話・シナリオ会話への適用チェックリスト

他ページへ反映する際、以下を確認する。

- [ ] エントリー: 一覧を最初に表示せず、ラージポップアップで選択→即開始する流れにできるか？
- [ ] 学習中: 「〇〇してください」などの行動要請テキストを削除できるか？
- [ ] 学習中: 表示エリアを固定し、テキストが消えていく演出にできるか？
- [ ] 学習中: 音声再生中を示すアニメーションアイコンを導入できるか？
- [ ] ListenFirst 系ポップアップ: 初回のみ・「もう1セット」時は出さない・永続非表示オプションを設けられるか？
- [ ] 一覧: テキストタップで再生開始（お気に入り以外）にできるか？
- [ ] 余白・バランス: 要素の位置がコンテンツ量でずれないか？
