# シーン別会話パターン数マトリックス

初級（1000語）および中級（3000語）における各シーンの目標パターン数と進捗管理用です。
`3分ストーリー作成内容まとめ` のシーン構成に準拠。

## パターンのバリエーション例（カフェ）

初級 5〜10 パターンの構成例:
- パターン1: カウンターでコーヒー注文
- パターン2: テイクアウト vs イートイン
- パターン3: メニューにないものを聞く
- パターン4: 注文の間違いを伝える
- パターン5: 会計・チップ
- パターン6: 電源・Wi-Fi について聞く
- パターン7: アレルギー対応を確認
- パターン8: 予約の問い合わせ

中級 15〜30 では、上記に加え:
- 支払い分割の相談
- ギフトカードの購入
- クレーム（温度・量）の丁寧な伝え方
- モーニングセット・ランチ特典の確認
など、シチュエーションを細分化。
